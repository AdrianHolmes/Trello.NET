# Trello.NET
A .NET client library for [Trello](https://trello.com).

[Getting started](https://github.com/dillenmeister/Trello.NET/wiki).

###Currently supports:
* Members (read only)
* Boards
* Lists
* Cards
* Checklists
* Organizations
* Notifications
* Tokens
* Actions
* Search
* Both sync and async APIs
