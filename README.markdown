# Trello.NET
A .NET client library for [Trello](https://trello.com).

[Getting started](https://github.com/dillenmeister/trellonet/wiki)

###Currently supports:
* Members (read only)
* Boards
* Lists
* Cards
* Checklists
* Organizations
* Notifications
* Tokens

###Currently does not support:
* Updating members
* OAuth
* Actions
